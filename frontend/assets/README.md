# Assets

Esta carpeta debe contener los siguientes archivos de imagen para la app:

- `icon.png` - Ícono de la app (1024x1024 px)
- `splash.png` - Pantalla de inicio (1242x2436 px)
- `adaptive-icon.png` - Ícono adaptativo Android (1024x1024 px)
- `favicon.png` - Favicon para web (48x48 px)

## Crear íconos temporales

Si no tienes íconos personalizados, puedes usar íconos por defecto de Expo o crear unos simples con cualquier herramienta de diseño.

Para desarrollo, Expo usará íconos por defecto si estos archivos no existen.

## Recomendaciones

- Usar formato PNG con transparencia
- Mantener diseño simple y reconocible
- Probar en diferentes tamaños de pantalla
- Usar colores que contrasten con fondos claros y oscuros
